# Pin All Dependency Versions

**TaskID**: XI41
**Status**: draft
**Order**: ⑥ — dead last, after every other dependency task
**Workflow**: `aexecute` directly — mechanical find-and-replace on package.json

## Elevator's Pitch

After all dependency updates are complete, convert every `^` and `~` range in package.json to exact version pins for deterministic builds. Update the `overrides` section to match.

## Business Gain

Deterministic builds. No surprise version bumps from semver ranges. Every `npm install` produces identical node_modules.

## Current State

Mix of exact pins (react, react-dom, next) and caret ranges (everything else) in package.json.

## Desired State

- Every dependency in package.json uses an exact version (no `^`, `~`, or `*`)
- `overrides` section matches pinned versions
- `npm run qa` passes

## Definition of Success

- Zero semver ranges in package.json
- `npm run qa` passes

## Additional Context

Run after all other update tasks are complete. This is a pure lockdown step — no version changes, just removing the `^` prefixes so future `npm install` doesn't silently bump anything.

## Constraints

- All other update tasks must be complete first
