**TaskID**: WP85
**Status**: draft

## Elevator's Pitch

Bump lucide-react from 0.x to 1.x. The v1.0 release removed brand icons (Github, Figma, Slack, etc.) but the project only uses generic icons — confirmed safe by codebase audit.

## Business Gain

Gets off the 0.x pre-release train onto a stable major version. Future minor/patch updates will be within a stable API.

## Rollback & Investigation Strategy

If icon errors occur after upgrade:
1. Check the error — is it a missing icon name (common after breaking changes)?
2. Cross-reference the icon used against [lucide v1.0 changelog](https://github.com/lucide-icons/lucide/releases/tag/0.263.1) for renamed/removed icons
3. Update the import to match the new icon name or find a suitable v1.x equivalent
4. Verify via `npm run qa` and manual testing
5. If a large number of icons are affected (unlikely per audit), consider downgrading and opening a task to audit custom icon usage
