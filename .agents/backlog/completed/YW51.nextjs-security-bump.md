# Bump Next.js to 16.2.4 — Security Fix

**TaskID**: YW51
**Status**: completed
**Order**: ② — after PE62, parallel with DQ53/VD22/KB55. High priority (security).
**Workflow**: `aexecute` directly — change 2 pins, npm install, validate

## Elevator's Pitch

Bump the Next.js exact pin from 16.1.6 to 16.2.4 to resolve 6 high-severity security vulnerabilities: HTTP request smuggling, CSRF bypass, disk cache DoS, postponed resume buffering DoS, and Server Component DoS.

## Business Gain

Eliminates 6 high-severity CVEs. The `next` package is the single largest source of audit findings in the project.

## Current State

- `next` pinned to exact `16.1.6` in package.json
- `npm audit` reports 6 high-severity advisories for next 16.0.0-beta.0 through 16.2.2:
  - GHSA-ggv3-7p47-pfv8: HTTP request smuggling in rewrites
  - GHSA-3x4c-7xq6-9pq8: Unbounded next/image disk cache growth
  - GHSA-h27x-g6w4-24gq: Unbounded postponed resume buffering DoS
  - GHSA-mq59-m269-xvcx: null origin CSRF bypass on Server Actions
  - GHSA-jcc7-9wpm-mj36: null origin HMR websocket CSRF bypass
  - GHSA-q4gf-8mx6-v5v3: Server Components DoS

## Desired State
