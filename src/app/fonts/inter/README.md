# Inter Font

This folder vendors Inter from the Google Fonts repository so production builds and browser sessions do not depend on Google Fonts network access.

## Files

- `Inter-Variable.ttf` — normal variable font, weights 100-900.
- `Inter-Italic-Variable.ttf` — italic variable font, weights 100-900.
- `OFL.txt` — SIL Open Font License 1.1 for Inter.
- `METADATA.pb` — Google Fonts metadata for the vendored family.

## Source

- Font source: https://github.com/google/fonts/tree/main/ofl/inter
- License source: https://github.com/google/fonts/blob/main/ofl/inter/OFL.txt
