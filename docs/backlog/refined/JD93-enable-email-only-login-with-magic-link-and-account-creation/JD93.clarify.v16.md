# Clarify Round v16 — Enable Email-Only Login With Magic Link and Account Creation

## Questions

### 1. What keys should throttle email requests and resends: app+email only, app+email+IP, or app+email+IP+user-agent?

## Answers

### 1.

For v1, throttle by `app+email` only.

Keep the data structure open so future iterations can add IP and possibly user-agent when needed.

## Resolution Summary

- V1 throttle key is app ID plus normalized email.
- V1 does not track IP or user-agent for throttling.
- Throttle storage/config should remain extensible for future IP and user-agent dimensions.
