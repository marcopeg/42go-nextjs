# Clarify Round v2 — Enable Email-Only Login With Magic Link and Account Creation

## Questions

### 1. What default manual code format do you want: 6-digit numeric, 8-character alphanumeric, or full verification token?

## Answers

### 1.

Default: 6 digits.

The code format should be configurable. A possible model is `length=6` plus `dict=0-9`, where `dict` acts like a regexp-style dictionary of available symbols. Naming and strategy can be refined; alternatives such as `alphabet`, `charset`, or presets should be considered.

## Resolution Summary

- Default manual code format is a 6-digit numeric code.
- Code generation must be configurable by length and allowed symbol set.
- Refinement should propose clear configuration names and validation strategy; `dict` is acceptable as an idea, but not yet final naming.
