# Clarify Round v10 — Enable Email-Only Login With Magic Link and Account Creation

## Questions

### 1. Should the `console` sender be blocked in production by default, or allowed in production only when explicitly selected?

## Answers

### 1.

Allowed. If the engineer configures `console` in production by mistake, that is an operator/configuration problem.

## Resolution Summary

- `console` sender may run in production when selected by configuration.
- The implementation should not block `console` solely because `NODE_ENV=production`.
- Documentation should make the production risk clear.
