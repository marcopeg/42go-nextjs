# Clarify Round v12 — Enable Email-Only Login With Magic Link and Account Creation

## Questions

### 1. Should v1 include both `console` and SMTP/Nodemailer sender strategies, or only `console` plus the strategy interface?

## Answers

### 1.

V1 should include `console` plus the first suggested external provider.

Chosen first external provider: Resend. Reason: official docs show a free transactional plan, a simple developer-focused HTTP API, and Node/Next.js documentation. `console` remains the default selected strategy; Resend is the first external strategy available for real delivery.

## Resolution Summary

- V1 includes both `console` and Resend sender strategies.
- `console` remains the default selected strategy.
- Resend is the first external email provider strategy.
- SMTP/Nodemailer can remain a future sender strategy unless planning decides to include it too.
