# Plan — Design LingoCafe Events Analytics Export Skill

## Goal

Create a download-only LingoCafe events export skill that reads new `lingocafe.events` rows from PostgreSQL via `LC_EVENTS_DATABASE_URL`, writes paired CSV and Parquet batch files into a gitignored local archive, advances a durable cursor only after validation, and documents the decision and operating workflow in `VI41.output.md`.
