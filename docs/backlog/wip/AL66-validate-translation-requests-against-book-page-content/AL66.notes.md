## Manual SQL

Remove only the stale `translate: true` key from users that currently have it, while preserving any other existing feature flags:

```sql
UPDATE auth.users
SET feature_flags = feature_flags - 'translate'
WHERE feature_flags @> '{"translate": true}'::jsonb;
```

```sql
SELECT id, app_id, email, feature_flags
FROM auth.users
WHERE feature_flags ? 'translate';
```
