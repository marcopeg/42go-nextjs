# Plan — Promote Events to Core App-Scoped Event System

## Goal

Replace the LingoCafe-only event stream with a core app-scoped events system: a new `events.events` hot-buffer schema, server and client logging APIs, per-app AppConfig controls, core user/profile/consent lifecycle event recording, LingoCafe write-point migration, and renamed export/logging skills that document and export the new structure.
