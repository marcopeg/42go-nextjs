## Questions

### 1. How should the fallback unread book be selected?

Recommended default: choose the first unread book matching the user's LingoCafe target language and target level from `lingocafe-users.parquet` when available, ordered by book `published_at` descending and then `title`. If profile language/level is missing or no exact match exists, fall back to the newest unread book.

Answer:
Accepted, with an expanded fallback chain.

For users without an active incomplete book, choose an unread book in this order:

1. Same target language and same target level, ordered by newest `published_at` and then `title`.
2. Same target language at the nearest higher available level.
3. Random unread book in the same target language.
4. If no unread book exists in the user's target language, skip the user and do not send an email.
