# Plan — LingoCafe Daily What To Read Next Email

## Goal

Implement `42go email lingocafe read-tip` as a local CLI automation that refreshes 42Go data, computes LingoCafe reading reminders from local Parquet files, dry-runs by default, sends through the existing Resend setup only when explicitly enabled, and records sent-email history for cooldown enforcement.
