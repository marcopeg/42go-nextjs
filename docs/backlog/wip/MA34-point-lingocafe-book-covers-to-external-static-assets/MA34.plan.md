# Plan — Point LingoCafe Book Covers to External Static Assets

## Goal

Replace LingoCafe's local book-cover URL resolution with centralized external asset URLs in the form `{LC_ASSETS_BASE_PATH}/{project}/reader`, default `LC_ASSETS_BASE_PATH` to `https://assets.lingocafe.app`, keep the generic placeholder fallback, and preserve existing catalog, detail, and reader behavior.
