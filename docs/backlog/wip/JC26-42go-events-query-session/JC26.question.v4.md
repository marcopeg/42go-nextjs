## Questions

### 1. Should `--limit` apply after filters?

Example: with `--user-id john --limit 20`, should it show the last 20 sessions for John?

Answer:
Yes. `--user-id`, `--app-id`, and `--limit` control console output only. Data processing should compute all available events into local aggregate cache files unless `--reset` is passed, which forces a full reflow from all available events into a fresh local aggregate cache.
