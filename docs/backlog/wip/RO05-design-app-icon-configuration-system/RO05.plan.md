# Plan - Design App Icon Configuration System

## Goal

Implement a shared, resolver-driven app icon system for the multi-app runtime. App-specific favicons, Apple touch icons, manifest icons, maskable icons, and UI title icons should resolve from `public/app-icons/<app-id>/`, fall back asset-by-asset to `public/app-icons/_default/`, and replace legacy `logo` and `public.pwa.icons` usage.
