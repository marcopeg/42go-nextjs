# quicklist - support edit task title in task list UI [acu]

Add support for editing a task's title directly in the quicklist task list UI.

## Goals

- [ ] User can edit a task title inline
- [ ] UI updates instantly on edit
- [ ] API call updates backend title
- [ ] Error handling for failed updates

## Acceptance Criteria

- [ ] Inline input for editing title
- [ ] Save/cancel controls
- [ ] API PATCH to update title
- [ ] Optimistic UI update
- [ ] Error message on failure
