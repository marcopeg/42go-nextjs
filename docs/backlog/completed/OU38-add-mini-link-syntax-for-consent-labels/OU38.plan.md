# Plan — Add Mini Link Syntax for Consent Labels

## Goal

Add one narrow consent-label link mechanic: string labels may wrap exactly one segment in `{{...}}`, and `labelLink` provides the anchor destination plus optional target defaulting to `_blank`. Existing plain labels, React-component labels, consent storage, and evidence statements stay unchanged.
