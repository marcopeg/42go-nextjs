# Execution Notes - Add 42go Backup and Restore CLI Commands
**Task**: ./GS88.task.md
**Plan**: ./GS88.plan.md

## Decisions

- Move backup and restore behavior into the Python `42go` CLI package.
- Keep `.local/42go-backups` as the default dump directory.
- Keep Makefile targets as direct `42go` wrappers with no fallback.
- Keep the old Node backup script as a compatibility shim that delegates to `42go`.

## Problems Encountered

- The active Python environment did not have `pytest` installed. Installed the editable CLI package with dev dependencies using `python3 -m pip install -e '.[dev]'`.
- The installed `42go` script landed in `/Library/Frameworks/Python.framework/Versions/3.13/bin`, which is not on this shell's PATH. Direct CLI verification used that full path.
