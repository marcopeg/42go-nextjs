## Questions

### 1. What should reset clear?

Choose the behavior that matches the intended "default reading experience":

- A: Reset only the active theme profile (`light`, `dark`, or `system`), matching the current reset scope.
- B: Reset the active theme profile and the shared font-size override.
- C: Reset all reader preferences across all theme profiles, including shared font size.

Answer:

You don't need to change the behavior. The behavior of the cleaning is okay. You have to change the user experience and the UI.

### 2. How should the reset action behave when the current reader settings are already default?

Choose one:

- A: Keep the destructive reset button visible but disabled.
- B: Hide the reset button when there is nothing custom to reset.
- C: Keep it enabled, but make it a harmless no-op without confirmation.

Answer:

Hide it. If there is no change from the current settings, from the default, then it shouldn't be even visible.

### 3. What exactly counts as a "custom reading experience" for the confirmation?

Choose one:

- A: Any saved reader preference data exists in local storage for the current scope.
- B: The currently visible preferences differ from computed defaults.
- C: Any reader preference data exists anywhere in the reader preference store.

Answer:

a
