# Plan — Reduce LingoCafe Route Transition Flicker With Delayed Loading State

## Goal

Introduce one reusable delayed-loading pattern for LingoCafe route transitions so `/books`, `/books/[bookId]`, and `/books/[bookId]/[pageId]` keep the current page visible during fast in-app navigations, only show loading after `0.8s`, and keep any shown loading state visible for at least `0.8s`, without affecting first direct load or refresh.
