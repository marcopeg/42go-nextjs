# Execution Notes — Build pipx-installable 42Go Events CLI
**Task**: ./AN44.task.md
**Plan**: ./AN44.plan.md

## Decisions

- Use `fortytwogo_cli` as the Python import package name because module names cannot start with a digit. Keep distribution name `42go-cli` and console script `42go`.
- Keep old event scripts only as delegating compatibility shims after the CLI owns the implementation.
- Add `pytz` as an explicit CLI dependency because global pipx validation with Python 3.13 showed DuckDB needed it when reading timezone-aware Parquet timestamps.

## Problems Encountered

- Sandbox DNS cannot resolve the Neon host in `EVENTS_DATABASE_URL`, so read-only `42go events pull --dry-run --limit 1` validation required approved network access.
- `compileall` could not create `__pycache__` under `.agents/skills/42go-events-export/scripts`; syntax was validated with in-memory `compile()` instead.

## Deviations From Plan

- Kept old event scripts as compatibility wrappers, but all implementation logic now lives in `cli/src/fortytwogo_cli/events/`.
