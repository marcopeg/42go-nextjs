### 2. Which desktop presentation API do you prefer?

Proposed option A:

```tsx
<Modal
  open={open}
  onOpenChange={setOpen}
  title="Preferences"
  presentation="panel"
  anchor="right"
  size="md"
>
  Body content
</Modal>
```

In this model, mobile is always full-screen; `presentation="modal"` means centered desktop modal; `presentation="panel"` means anchored desktop side panel.

Alternative option B:

```tsx
<Modal
  open={open}
  onOpenChange={setOpen}
  title="Preferences"
  desktop={{ presentation: 'panel', anchor: 'right', size: 'md' }}
>
  Body content
</Modal>
```

Answer:

I like version A.
