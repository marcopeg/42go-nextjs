# Plan — Improve LingoCafe Book Card Image-First Design

## Goal

Turn the LingoCafe `/books` catalog card into an image-first, cover-like card: no visible border or metadata panel, overlaid book-cover title near the top, max three overlaid tags near the bottom, preserved `2:3` ratio, and at least two covers per row on mobile.
