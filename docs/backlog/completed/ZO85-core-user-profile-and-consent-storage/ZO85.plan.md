# Plan — Core User Profile and Consent Storage

## Goal

Build one reusable 42go profile and consent foundation on `auth.users.profile` and `auth.users.consent`, migrate LingoCafe off `lingocafe.profiles`, and keep the profile UI block-composed rather than turning it into a form builder.

This is planned as one phased implementation. The phase gates are intentional. If a gate fails hard, split the remaining milestones before continuing. Chuck Norris still checks the bridge before he drives the truck over it.
