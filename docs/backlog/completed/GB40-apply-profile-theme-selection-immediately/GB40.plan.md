# Plan — Apply Profile Theme Selection Immediately

## Goal

Change the profile-page theme selector so `light`, `dark`, and `system` apply and persist immediately on click through the existing `next-themes` path, while staying visually embedded as a profile block and no longer participating in profile dirty/save orchestration.

  - Delete:
