## Decisions

- Treat the operator's "yes and execute" as plan acceptance plus execution approval.
- Move `MA34` from Drafts to WIP before code changes.
- Keep `/images/lingocafe/placeholder.jpg` as the generic fallback while replacing local book-specific cover URLs with external project-based URLs.
- Centralize the new external cover URL in `src/app/api/(lingocafe)/lingocafe/_lib/reader.ts` so catalog, detail, and reader payloads use the same resolver.
- Normalize `LC_ASSETS_BASE_PATH` by trimming whitespace and trailing slashes before joining it with `{project}/reader`.
