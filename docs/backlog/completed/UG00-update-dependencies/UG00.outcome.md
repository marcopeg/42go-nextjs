## Proposed Update Roadmap

The following tasks can be created as independent backlog items. They're ordered from safest to riskiest. Tasks within the same tier can run in parallel.

### Task A — Fix Stale Lockfile + Remove Dead Deps

**Risk**: Low-Medium  
**Effort**: 15 min  
**Priority**: Do first — everything else depends on this

Steps:
1. `npm uninstall oci-objectstorage oci-secrets @azure/app-configuration @azure/keyvault-secrets`
2. `npm install` (resolves all stale lockfile entries)
3. `npm run qa`
4. Test dev server startup and basic auth flow

**Why first**: The lockfile is broken. Every other task will trigger these stale resolutions anyway. Better to do it deliberately and validate once.

**Risk notes**: This will pull in @dnd-kit/sortable 10.x, uuid 13.x, react-syntax-highlighter 16.x, dotenv 17.x, @types/bcrypt 6.x, @types/node 25.x, eslint-config-next 16.x all at once. The package.json already declares these ranges, so this is catching up to what was already "approved" but never installed. The dnd-kit jump (8→10) is the one to watch — test QuickList drag-and-drop after.
