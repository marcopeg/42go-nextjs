# Clarify Round v1 — Replace Books Filter Text With Round Language Flag

## Questions

### 1. Should the round flag be purely visual in this task, or should tapping it already open the future language switcher?

### 2. If there is no active learning language, what should the header show instead: nothing, a neutral placeholder, or the existing fallback text?

### 3. Should the flag sit on the same top row as `Books`, aligned to the right edge, or lower where the subtitle currently sits?

## Answers

### 1.

Keep it as a simple visual flag. Tapping it can route to the profile page.

### 2.

Show nothing when there is no active learning language. That means the user is still completing the profile.

### 3.

Place the flag on the same header row, aligned to the right edge and vertically centered.

## Resolution Summary

- The flag stays lightweight in this task and does not open a switcher or popup.
- Tapping the flag routes to `/profile`.
- If there is no active learning language, the header shows no replacement badge.
- The flag sits on the `Books` header row, right-aligned and vertically centered.
