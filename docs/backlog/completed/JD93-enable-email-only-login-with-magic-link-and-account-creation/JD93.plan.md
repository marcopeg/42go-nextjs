# Plan — Enable Email-Only Login With Magic Link and Account Creation

## Goal

Implement app-configurable passwordless email authentication through NextAuth v4 EmailProvider and a custom app-scoped Knex adapter, preserving existing credentials/OAuth behavior while adding magic links, manual code entry, sender strategies, throttling, audit events, docs, and completion evidence.
