# Plan — Investigate and Optimize Docker Build Performance

## Goal

Measure the current Docker build cost, then implement safe local-first and VPS-focused publish speedups. The clean no-cache path stays available. The default publish path becomes the practical VPS path, not the heavyweight universal build.
