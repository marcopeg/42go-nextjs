# Plan — Add Makefile Database Backup and Restore Commands

## Goal

Add Makefile-driven PostgreSQL data backup and restore commands backed by a repo-local `42go-backup` skill. Backups produce self-contained UTC-stamped SQL files under `knex/dumps/`, with dependency-safe truncate and insert instructions that can be restored after migrations have created the schema.
