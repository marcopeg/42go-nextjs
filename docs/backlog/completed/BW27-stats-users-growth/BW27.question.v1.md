### 4. Which events count as reading activity?

Recommended: count `page.open`, `page.scroll`, and `page.translate`; exclude settings-only events like `read.settings.opened` and `read.settings.changed`.

Answer:

go with your recommended, sample the data for meaningful events.

### 5. How should weekly active users be computed?

Recommended: trailing 7-day window ending at each bucket boundary. Example: the daily point for 2026-06-10 counts users with reading events from 2026-06-04 through 2026-06-10.

Answer:

ok for me.

### 6. How should monthly active users be computed?

Recommended: trailing 30-day window ending at each bucket boundary, not calendar-month-only activity.

Answer:

ok for me.

### 7. How should inactive users be computed?

Recommended: known users at the bucket boundary minus monthly active users at that same boundary.

Answer:

good for me.
